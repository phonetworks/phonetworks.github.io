\Pho\Lib\Graph

\Pho\Lib\GraphID

Immutable, cryptographically secure identifier

Pho IDs are immutable and come in the format of cryptographically secure UUIDv4

Pho IDs are used to define all graph entities, e.g nodes and edges.

The only ID that doesn't conform with the UUID format is the Graph ID which is by default set to be a period (.) and it may be called as ID::root()

Even at scale of billions of nodes and edges, the chances of collision is identical to zero.

You can generate a new ID with $id_object = ID::generate() and fetch its string representation with PHP type-casting; (string) $id_object.
